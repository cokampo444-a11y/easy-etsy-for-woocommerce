# Etsy OAuth Redirect für Easy Etsy for WooCommerce

## Zweck
Diese App fängt OAuth-Callbacks von Etsy ab und leitet sie an deine WooCommerce-Installation weiter.

## Datenschutz
- **Keine Datenspeicherung**: Codes werden sofort weitergeleitet, nicht gespeichert
- **Keine Cookies**: Es werden keine Cookies gesetzt
- **Kein Tracking**: Keine Analytics oder Tracking-Tools
- **Logs**: Vercel speichert automatisch Logs für 30 Tage (IPs anonymisiert)
